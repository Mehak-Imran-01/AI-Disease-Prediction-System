# AI-Disease-Prediction-System

## Overview
An AI-powered web application that predicts diseases based on user symptoms. 
The system integrates a machine learning model with a Flask backend and a user-friendly web frontend. 
MongoDB is used to store user data and prediction history.

## Features
- Predicts diseases using ML models
- User-friendly web interface
- Stores patient information and predictions in MongoDB
- Provides prediction confidence scores

## Tech Stack
- Python, Flask, Scikit-learn, Pandas
- MongoDB for database
- HTML, CSS, JavaScript for frontend
- XGBoost for ML prediction

## Install python Dependencies
pip install -r backend/requirements.txt

## Run Flask Backend
python backend/app.py

## Run Frontend
Open frontend/index.html in a browser to use the app
